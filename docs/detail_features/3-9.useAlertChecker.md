# useAlertChecker: 3종 알림 실시간 감지

> 📄 **상위 문서**: [ETF_DETAIL.md](./2.ETF_DETAIL.md) § 3-9

종목 상세 페이지가 열려 있는 동안 **실시간으로 3종 알림**을 감지하는 훅. 목표가 도달·급등/급락·매매 시그널을 판별하고, 토스트·AlertContext·백엔드 트리거 기록을 수행한다.

---

## 1. 동작 방식 개요

- **호출**: ETFDetail에서 1회, ticker·alertRules·intradayData·previousClose·tradingFlowData 전달
- **부수 효과**: 토스트 표시, AlertContext.pushAlert, POST /api/alerts/trigger
- **반환값**: 없음 (void 훅)
- **중복 방지**: `firedRef`(Set), `lastTradingFlowDateRef`로 세션 내 재발생 차단

---

## 2. 감지 대상 및 로직

### 2-1. 목표가 도달 (useEffect: intradayData, alertRules)

| 규칙 | 조건 | hit |
|------|------|-----|
| buy + below | currentPrice ≤ target | 알림 |
| buy + above | currentPrice ≥ target | 알림 |
| sell + above | currentPrice ≥ target | 알림 |
| sell + below | currentPrice ≤ target | 알림 |

- `currentPrice`: intradayData.data[마지막].price
- 대상: alert_type ∈ { buy, sell }, is_active === true

### 2-2. 급등/급락 (useEffect: intradayData, previousClose, alertRules)

| 규칙 direction | 조건 | hit |
|---------------|------|-----|
| above | changePct ≥ threshold | 알림 |
| below | changePct ≤ -threshold | 알림 |
| both | \|changePct\| ≥ threshold | 알림 |

- `changePct` = (currentPrice - previousClose) / previousClose * 100
- `threshold` = rule.target_price (임계 %)
- 대상: alert_type === 'price_change', is_active

### 2-3. 매매 시그널 (useEffect: tradingFlowData, alertRules)

| 조건 | hit |
|------|-----|
| foreign_net > 0 && institutional_net > 0 | "외국인+기관 동시 순매수" |
| foreign_net < 0 && institutional_net < 0 | "외국인+기관 동시 순매도" |

- `latest` = tradingFlowData[0] (최신 1일)
- **날짜별 중복 방지**: `lastTradingFlowDateRef`에 dateKey 저장, 같으면 스킵
- 대상: alert_type === 'trading_signal', is_active

---

## 3. fireAlert 헬퍼

```javascript
fireAlert(rule, message, toastType)
```

1. `key = ${rule.id}-${rule.alert_type}` → firedRef에 있으면 return
2. firedRef.add(key)
3. toast[toastType](message, 5000)
4. alertStore.pushAlert({ ticker, tickerName, alert_type, message })
5. alertApi.recordTrigger({ rule_id, ticker, alert_type, message }).catch(() => {})
```

---

## 4. 종목 변경 시 초기화

```javascript
useEffect(() => {
  firedRef.current = new Set()
  lastTradingFlowDateRef.current = null
}, [ticker])
```

---

## 5. 토스트 타입

| 알림 | toastType |
|------|-----------|
| buy 목표가 | info |
| sell 목표가 | success |
| 급등 | warning |
| 급락 | error |
| 동시 순매수 | success |
| 동시 순매도 | error |

---

## 6. 문제 해결

| 현상 | 원인 | 조치 |
|------|------|------|
| 목표가 알림 안 옴 | intradayData 없음 또는 alertRules 비활성 | 분봉 로드, is_active 확인 |
| 급등/급락 안 옴 | previousClose null | pricesData 2건 이상 확인 |
| 매매 시그널 중복 | lastTradingFlowDateRef 미작동 | 같은 dateKey 스킵 로직 확인 |
| 토스트만 되고 벨에 안 뜸 | alertStore.pushAlert 실패 | AlertContext 연결 확인 |
| 트리거 기록 실패 | API 에러 | recordTrigger catch 로그, 백엔드 확인 |
