# PriceTable: 가격 데이터 테이블

> 📄 **상위 문서**: [ETF_DETAIL.md](./2.ETF_DETAIL.md) § 3-11

일자별 **OHLC, 거래량, 등락률**을 테이블로 표시. 접기/펼치기(기본 접힘), 페이지당 20건, 정렬·페이지네이션 지원. 반응형(모바일 카드 형태), 다크모드 지원.

---

## 1. 동작 방식 개요

- **데이터**: ETFDetail의 pricesData (추가 API 없음)
- **위치**: ETFDetail 하단, 접기/펼치기 버튼이 있는 카드 내부
- **기본 상태**: `isTableExpanded` false → 테이블 본문 숨김
- **버튼**: "가격 데이터 (N건)" + "펼치기 ▼" / "접기 ▲"

---

## 2. 테이블 컬럼

| 컬럼 | 데이터 키 | 정렬 키 | 포맷 |
|------|----------|---------|------|
| 날짜 | date | date | yyyy-MM-dd |
| 시가 | open_price | — | formatPrice |
| 고가 | high_price | — | formatPrice |
| 저가 | low_price | — | formatPrice |
| 종가 | close_price | close | formatPrice |
| 거래량 | volume | volume | formatVolume |
| 등락률 | daily_change_pct | change | formatPercent, getPriceChangeColor |

---

## 3. 정렬

- **초기**: date desc (최신순)
- **핸들러**: handleSort(key) — 동일 키 클릭 시 asc↔desc 토글
- **지원 키**: date, close, volume, change
- **정렬 시**: currentPage 1로 리셋

---

## 4. 페이지네이션

- **itemsPerPage**: 기본 20
- **totalPages**: Math.ceil(sortedData.length / itemsPerPage)
- **currentPage**: useState, 페이지 전환 버튼으로 변경
- **표시**: "1 / N" 형태, 이전/다음 버튼

---

## 5. 반응형

- **데스크톱**: 테이블 형태
- **모바일**: 카드 형태 (각 row를 카드로, flex/stack 배치)

---

## 6. 데이터 요구사항

pricesData 각 항목:

```javascript
{
  date: string,           // yyyy-MM-dd
  open_price: number,
  high_price: number,
  low_price: number,
  close_price: number,
  volume: number,
  daily_change_pct: number
}
```

---

## 7. ETFDetail 연동

- **접기 상태**: `isTableExpanded` (useState, 기본 false)
- **버튼**: onClick={() => setIsTableExpanded(!isTableExpanded)}
- **조건부 렌더**: isTableExpanded일 때만 `<PriceTable data={pricesData} itemsPerPage={20} />` 렌더

---

## 8. 문제 해결

| 현상 | 원인 | 조치 |
|------|------|------|
| 테이블 비어 있음 | pricesData 빈 배열 | 가격 API, 날짜 범위 확인 |
| 정렬 후 순서 이상 | sortConfig key 오타 | date/close/volume/change 확인 |
| 등락률 색상 안 나옴 | getPriceChangeColor 미적용 | td에 해당 클래스 적용 확인 |
| 모바일에서 레이아웃 깨짐 | 카드 형태 미구현 | 모바일 breakpoint에서 카드 스타일 적용 |
