# ETFHeader: Sticky 헤더

> 📄 **상위 문서**: [ETF_DETAIL.md](./2.ETF_DETAIL.md) § 3-1

종목 상세 페이지(`/etf/:ticker`) 최상단에 배치되는 **Sticky 헤더**. 스크롤 시에도 항상 상단에 고정되어 종목명·티커·테마를 표시하며, 다크모드를 지원한다.

---

## 1. 동작 방식 개요

- **위치**: `position: sticky`, `top: 0`, `z-index: 50`으로 뷰포트 상단 고정
- **데이터**: 부모(ETFDetail)에서 `GET /api/etfs/{ticker}` 응답의 `etf` 객체 전달
- **렌더**: API 호출 없이 props만으로 렌더링 (순수 프레젠테이션 컴포넌트)

### 레이어 구조

```
┌─ 배경 레이어 (absolute inset-0) ─────────────────┐
│  bg-white/dark:bg-gray-800, border-b, shadow-sm   │
│  -mx-4 sm:-mx-6 lg:-mx-8 (카드 패딩 보정)          │
└──────────────────────────────────────────────────┘
┌─ 내용 레이어 (relative py-4) ────────────────────┐
│  h1: 종목명 (text-3xl font-bold)                 │
│  p:  티커 · 테마 (text-gray-600)                  │
└──────────────────────────────────────────────────┘
```

---

## 2. 표시 항목 상세

| 항목 | 소스 | 조건 | 포맷 |
|------|------|------|------|
| 종목명 | `etf.name` | 없으면 "ETF 상세" | h1, text-3xl |
| 티커 | `etf.ticker` | 항상 | `{ticker} · {theme}` |
| 테마 | `etf.theme` | 항상 | 동일 라인에 표시 |

### 다크모드

- `dark:bg-gray-800`: 배경
- `dark:text-gray-100`: 제목
- `dark:text-gray-400`: 부제(티커·테마)
- `dark:border-gray-700`: 하단 구분선

---

## 3. 구현 세부사항

### PropTypes

```javascript
etf: PropTypes.shape({
  name: PropTypes.string,
  ticker: PropTypes.string,
  theme: PropTypes.string,
})
```

### date-fns 사용

- `format(etf.purchase_date, 'yyyy-MM-dd')` 등: ETFHeader 자체에는 미사용. etf 객체만 표시.

---

## 4. 문제 해결

| 현상 | 원인 | 조치 |
|------|------|------|
| 스크롤 시 헤더가 가려짐 | z-index 부족 | `z-50` 확인, 상위 요소 stacking context 검토 |
| 레이아웃 넘침 | 컨테이너 패딩 미보정 | `-mx-4 sm:-mx-6 lg:-mx-8`로 main 패딩 상쇄 |
| 다크모드 미적용 | Tailwind dark: 클래스 누락 | `dark:bg-gray-800` 등 dark: 접두사 확인 |
