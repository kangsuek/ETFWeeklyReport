# ETFCharts: 차트 영역

> 📄 **상위 문서**: [ETF_DETAIL.md](./2.ETF_DETAIL.md) § 3-6

가격, 매매 동향, RSI, MACD, 지지선/저항선을 **수직 배치**하는 통합 차트 컨테이너. 표시 조건·스크롤 동기화·기술지표 데이터 연동을 담당한다.

---

## 1. 동작 방식 개요

- **차트 4종**: PriceChart, TradingFlowChart, RSIChart, MACDChart
- **부가**: TechnicalIndicatorsSection (RSI+MACD), SupportResistanceSection (지지/저항)
- **표시 조건**: settings 및 토글로 제어
- **스크롤 동기화**: 가격 차트 ↔ 매매동향 차트 가로 스크롤

---

## 2. 차트별 표시 조건

| 차트 | 컴포넌트 | 조건 |
|------|----------|------|
| 가격 | PriceChart | `showVolume` (settings.display.showVolume, 기본 true) |
| 매매 동향 | TradingFlowChart | `showTradingFlow` (settings.display.showTradingFlow, 기본 true) |
| RSI | RSIChart | `showRSI` (useState, 기본 true) |
| MACD | MACDChart | `showMACD` (useState, 기본 true) |
| 기술지표 섹션 | TechnicalIndicatorsSection | `showTechnicalSection` (기본 true, 항상 표시) |
| 지지/저항 | SupportResistanceSection | supportResistanceData 있으면 표시 |

---

## 3. 가격 차트 부가 기능

- **매입가 수평선**: `purchasePrice` 전달 시 ReferenceLine 표시
- **지지선/저항선**: `calculateSupportResistance(pricesData)` — 피봇, 이동평균, 20일 고/저 기반

---

## 4. 스크롤 동기화

- **구현**: `priceChartScrollRef`, `tradingFlowChartScrollRef`로 두 차트의 scrollLeft 동기화
- **핸들러**: `onPriceChartScroll`, `onTradingFlowChartScroll` — 한 쪽 스크롤 시 다른 쪽에 반영
- **무한 루프 방지**: `isScrollingSyncRef` 플래그 + `requestAnimationFrame`으로 다음 프레임에서 플래그 해제

```javascript
const handlePriceChartScroll = () => {
  if (isScrollingSyncRef.current) return
  isScrollingSyncRef.current = true
  tradingFlowChartScrollRef.current.scrollLeft = priceChartScrollRef.current.scrollLeft
  requestAnimationFrame(() => { isScrollingSyncRef.current = false })
}
```

---

## 5. 기술지표 데이터

### 5-1. RSI/MACD 계산용 확장 데이터

- **필요**: RSI 14일 → 15건 이상, MACD 12/26/9 → 35건 이상. **선행 60일** 추가 요청
- **API**: `GET /api/etfs/{ticker}/prices` (startDate = dateRange.startDate - 60일)
- **enabled**: `showRSI || showMACD`일 때만 useQuery 실행
- **정렬**: API는 최신순 → 오름차순 변환 후 calculateRSI, calculateMACD

### 5-2. 지지/저항

- **계산**: `calculateSupportResistance(pricesData)` — pricesData 그대로(최신순)
- **반환**: currentPrice, pivot, movingAverages, recentLevels, supports, resistances

---

## 6. TechnicalIndicatorsSection 상세

- **RSI**: 14일, 해석 텍스트 (과매수/과매도/상승/하락)
- **MACD**: 12/26/9, 골든/데드크로스·모멘텀 해석
- **토글**: RSI, MACD 각각 체크박스로 표시/숨김
- **둘 다 꺼짐**: "RSI 또는 MACD를 선택하면 기술지표 차트가 표시됩니다" 안내

---

## 7. SupportResistanceSection 상세

- **저항선**: 상단, 빨간 막대·가격·거리%
- **현재가**: 가운데 강조 바
- **지지선**: 하단, 파란 막대·가격·거리%
- **최근 고/저**: 20일 최고가·최저가 카드
- **상세 토글**: "전체 레벨 보기" 펼치면 저항/지지 전체 목록 + 읽는 법

---

## 8. 로딩/에러

각 차트 영역별로 LoadingIndicator 또는 ErrorFallback, refetch 버튼 제공.

---

## 9. 문제 해결

| 현상 | 원인 | 조치 |
|------|------|------|
| RSI/MACD 안 나옴 | 확장 데이터 미요청 | showRSI/showMACD true 확인, extendedDateRange 계산 |
| 스크롤 동기화 안 됨 | ref 미전달 또는 scroll 이벤트 미연결 | PriceChart/TradingFlowChart에 scrollRef, onScroll 확인 |
| 지지/저항 빈 화면 | supportResistanceData null | calculateSupportResistance 호출 및 pricesData 전달 확인 |
| 무한 스크롤 루프 | isScrollingSyncRef 미사용 | requestAnimationFrame 내 플래그 해제 확인 |
