# StrategySummary: 투자 전략

> 📄 **상위 문서**: [ETF_DETAIL.md](./2.ETF_DETAIL.md) § 3-3

**백엔드 인사이트 API** 기반 투자 전략. 단기·중기·장기 전략과 종합 추천·코멘트, 핵심 포인트를 표시한다. 전략 유형별 뱃지 색상으로 시각 구분한다.

---

## 1. 동작 방식 개요

- **API**: `GET /api/etfs/{ticker}/insights?period={1w|1m|3m|6m|1y}`
- **기간 연동**: ETFDetail의 날짜 범위에 따라 period 매핑 (7d→1w, 1m→1m, 3m→3m)
- **캐시**: staleTime 2분 (CACHE_STALE_TIME_SLOW)
- **순서**: useQueries 2순위 (인사이트)로 병렬 호출

### 응답 구조 (기대)

```json
{
  "strategy": {
    "short_term": "비중확대|보유|관망|비중축소",
    "medium_term": "비중확대|보유|관망|비중축소",
    "long_term": "비중확대|보유|관망|비중축소",
    "recommendation": "비중확대|보유|관망|비중축소",
    "comment": "종합 코멘트 텍스트"
  },
  "key_points": ["포인트1", "포인트2", ...]
}
```

## 2. 전략 뱃지 색상

| 전략 유형 | Tailwind 클래스 | 용도 |
|----------|-----------------|------|
| 비중확대 | `bg-green-100 text-green-800 dark:bg-green-900/30 dark:text-green-400` | 매수 유도 |
| 보유 | `bg-blue-100 text-blue-800 dark:bg-blue-900/30 dark:text-blue-400` | 현상태 유지 |
| 관망 | `bg-gray-100 text-gray-800 dark:bg-gray-800 dark:text-gray-300` | 대기 |
| 비중축소 | `bg-red-100 text-red-800 dark:bg-red-900/30 dark:text-red-400` | 매도 유도 |

## 3. UI 구성

### 3-1. 투자 전략 섹션

- **단기·중기·장기**: 각각 뱃지로 표시
- **종합**: `strategy.recommendation` 뱃지 + `strategy.comment` 텍스트

### 3-2. 핵심 포인트 섹션

- `key_points` 배열이 있으면 `<ul>` 리스트로 표시
- 각 항목: 체크 아이콘 + 텍스트
- key_points가 없거나 빈 배열이면 섹션 자체 미표시

### 3-3. 상태별 처리

| 상태 | UI |
|------|-----|
| isLoading | Spinner + "인사이트 분석 중..." |
| error | 경고 카드 (노란 배경) + 에러 메시지 |
| !insights | null (미렌더) |

---

## 4. period 매핑

| dateRange.range | insightsPeriod | API period |
|-----------------|----------------|------------|
| 7d | 1w | 1w |
| 1m | 1m | 1m |
| 3m | 3m | 3m |

ETFDetail에서 `insightsPeriod`를 계산하여 StrategySummary에 전달.

---

## 5. 문제 해결

| 현상 | 원인 | 조치 |
|------|------|------|
| 전략이 표시되지 않음 | insights API 에러 또는 null | 에러 UI 확인, 백엔드 로그 확인 |
| period 불일치 | dateRange와 insightsPeriod 동기화 오류 | ETFDetail의 insightsPeriod useMemo 확인 |
| 핵심 포인트가 비어 있음 | API가 key_points 미반환 | 백엔드 insights_service 응답 스키마 확인 |
| 뱃지 색상 이상 | strategy 값이 4종 외 | getStrategyColor fallback → '관망' (회색) |
