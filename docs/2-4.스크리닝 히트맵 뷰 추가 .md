│ >= 3%  → #dc2626 (빨강)                                                                                              │

│ >= 0%  → #fca5a5 (아주 연빨강)                                                                                       │
│ >= -1% → #93c5fd (연파랑)                                                                                            │
│ >= -3% → #3b82f6 (파랑)                                                                                              │
│ >= -5% → #1d4ed8 (진파랑)                                                                                            │
│ < -5%  → #1e3a8a (아주 진파랑)                                                                                       │
│ null   → #9ca3af (회색)                                                                                              │
