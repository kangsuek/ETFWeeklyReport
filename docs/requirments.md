## 1단기: 실시간 / 준실시간 데이터

### 1-1. 실시간 / 준실시간 데이터

- 현재: 네이버 금융 스크래핑 (지연 데이터), 분봉은 장중 20초 갱신.

- 필요:

- 최소 준실시간(1~5분 지연) 가격 데이터

- 정규 데이터 제공업체 연동 (KRX 정보데이터, 한국투자증권 API 등)

- 웹스크래핑 의존 탈피 (상업화 시 법적 리스크)

- WebSocket 기반 실시간 가격 스트리밍 (프리미엄)

## 2단계: 경쟁력 확보 (경쟁 서비스와 비교 시 반드시 필요)

### 2-1. 알림 / 푸시 기능

- 현재: Toast 알림(화면 내)만 존재. 앱을 안 보면 모름.

- 필요:

- 목표가 알림: "종가가 N원 이상/이하일 때 알림"

- 급등/급락 알림: 일간 등락률 ±N% 초과 시

- 매매 시그널: 외국인·기관 동시 매수/매도 감지

- 뉴스 알림: 관련 키워드 뉴스 발생 시

- 채널: 브라우저 푸시, 이메일, 카카오 알림톡

### 2-2. 종목 발굴 / 스크리닝

- 현재: 등록된 종목만 분석 가능. 발굴 기능 없음.

- 필요:

- 테마별 탐색: "AI", "반도체", "2차전지" 등 테마별 ETF 목록

- 조건 필터: 수익률 상위, 변동성 하위, 외국인 순매수 상위 등

- 추천 종목: "이번 주 주목할 ETF", "외국인 매수 급증 ETF"

- 인기 종목: 다른 사용자들이 많이 추가한 종목

## 3단계: 차별화 / 가치 제고 (유료 전환율을 높이는 기능)

### 3-1. AI 기반 분석

- 현재: 룰 기반 인사이트 (수익률 >X% → "매수" 등 조건문).

- 필요:

- LLM 기반 종목 분석 요약 (뉴스 + 가격 + 수급 종합)

- "이 종목, 지금 사도 될까요?" 자연어 질의

- 포트폴리오 AI 코치: 리밸런싱 제안, 리스크 경고

- 뉴스 감성 분석 (긍정/부정/중립)

### 3-2. 백테스팅 / 투자 시뮬레이션 강화

- 현재: 비교 페이지에 100만원 시뮬레이션만 있음.

- 필요:

- 과거 특정 날짜에 매수했다면 수익률 시뮬레이션

- 적립식 투자 시뮬레이션 (매월 N만원씩)

- 포트폴리오 백테스팅 (비중 조합별 과거 성과)
