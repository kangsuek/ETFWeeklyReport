## 추가 최적화 방안

### 1. 문서 분리 전략

**현재 문제**: 모든 정보가 한 문서에 집중

**개선 방안**:
- ✅ **핵심 정보만** 각 문서에 유지
- ✅ **상세 내용은** 별도 문서로 분리 (예: `docs/detailed/`)
- ✅ **참조 링크** 중심으로 구조화

### 2. 중복 제거 전략

**중복 내용**:
1. 종목 목록 (CLAUDE.md, README.md) → **README.md에만 유지**
2. 프로젝트 현황 (CLAUDE.md, README.md) → **PROGRESS.md에만 유지**
3. 빠른 시작 (README.md, SETUP_GUIDE.md) → **README.md에만 유지**
4. 기술 스택 상세 (README.md, TECH_STACK.md) → **TECH_STACK.md에만 유지**

### 3. 참조 중심 구조

**원칙**:
- 각 문서는 **자신의 목적에만 집중**
- 상세 내용은 **다른 문서 참조**
- **"자세한 내용은 [문서명] 참조"** 형식 사용
