### 1. HTML 엔티티 디코딩

**기능**: HTML 엔티티를 실제 문자로 변환
- `&quot;` → `"`
- `&amp;` → `&`
- `&lt;` → `<`
- `&gt;` → `>`

**구현 방법**:
```python
import html

def clean_html_tags(text: str) -> str:
    clean_text = re.sub(r'<[^>]+>', '', text)
    clean_text = html.unescape(clean_text)  # 엔티티 디코딩
    return clean_text
```

**테스트 결과**:
```

변환: "사업형 투자회사로"

원본: AI &amp; 데이터센터
변환: AI & 데이터센터
```

✅ **성공**: 모든 HTML 엔티티가 정상적으로 디코딩됨

### 3. 관련도 점수 계산

**기능**: 키워드 기반 관련도 점수 산출 (0.0 ~ 1.0)

**계산 방식**:
- 제목에 키워드 포함: +1점
- 본문에 키워드 포함: +1점
- 정규화: 점수 / (키워드 수 × 2)

**구현 방법**:
```python
def calculate_relevance_score(news_item: Dict, keywords: List[str]) -> float:
    title = clean_html_tags(news_item['title']).lower()
    description = clean_html_tags(news_item['description']).lower()

    score = 0.0
    max_score = len(keywords) * 2

    for keyword in keywords:
        if keyword.lower() in title:
            score += 1.0
        if keyword.lower() in description:
            score += 1.0

    return score / max_score if max_score > 0 else 0.0
```

**테스트 결과** (AI 전력, 키워드: ['AI', '전력', '데이터센터']):

[2] 점수: 0.83 - 미쓰비시 중공업, 수주 전망 16% 상향…데이터센터 전력수요 급증
